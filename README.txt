---------------Welcome to the SpringBoot Facebook login demo---------------

1. Clone the roject using "https://github.com/shanakaweerasinghe/SpringBoot_SocialLogin.git"
2. Build the project using "mvn package" command.
3. Navigete to target directory of the roject and run the "spring-boot-social-login-0.0.1-SNAPSHOT.jar" file 
using "java -jar spring-boot-social-login-0.0.1-SNAPSHOT.jar" command.
4. Once applicatio is started, navigate to "http://localhost:8080" url with new browser window.
5. Click the login with FaceBook link to authentcate through Facebook and retrieve user information with obteined 
Access token.
